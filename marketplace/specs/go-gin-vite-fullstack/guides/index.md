# Guides

Use these guides to avoid design mistakes before writing code.

| Guide | Purpose | When to Use |
| --- | --- | --- |
| [Task Flow](./task-flow.md) | Default workflow | Any task |
| [Pre-Implementation Checklist](./pre-implementation-checklist.md) | Readiness questions before code | Non-trivial changes |
| [Cross-Layer Thinking Guide](./cross-layer-thinking-guide.md) | Go/frontend/container boundary thinking | Fullstack, middleware, config, or Docker changes |
| [Review Checklist](./review-checklist.md) | Final handoff check | Before completion |

## Core Habit

Keep request context and logging behavior intact while changing HTTP code.
